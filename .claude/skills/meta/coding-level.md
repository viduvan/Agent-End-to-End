# Skill: /fis:coding-level — Set Coding Level

> Đặt mức kinh nghiệm coding để tuỳ chỉnh output. Adjust agent behavior based on developer level.

## Khi nào dùng
- Set skill level cho personalized agent output
- Junior: more explanations, step-by-step
- Senior: concise, advanced patterns
- Expert: minimal explanation, cutting-edge approaches

## Agent
Meta (System)

## Prompt mẫu
```
/fis:coding-level "Set to senior - I want concise code with minimal comments"
```
```
/fis:coding-level "Set to junior - explain each step in detail"
```

- [ ] Agent output matches requested level
