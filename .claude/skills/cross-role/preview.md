# Skill: /fis:preview — Preview Generated Content

> Xem file hoặc sinh slide/diagram/giải thích trực quan. Visual preview generation.

## Khi nào dùng
- Preview generated content trước khi commit
- Sinh visual explanations (slides, diagrams)
- Quick visual check cho UI components

## Agent
UI/UX Designer, Technical Analyst

## Prompt mẫu
```
/fis:preview "Generate visual preview of this API response structure"
```
```
/fis:preview "Create slide deck summarizing this sprint's changes"
```

## Verification
- [ ] Preview renders correctly
- [ ] Content matches expectations
