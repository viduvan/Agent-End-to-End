# Skill: /fis:ask — Structured Question Asking

> Tư vấn câu hỏi kỹ thuật và kiến trúc với phân tích chuyên gia. Expert consultation.

## Khi nào dùng
- Câu hỏi kỹ thuật cần phân tích chuyên gia
- Architecture decisions cần second opinion
- Best practices cho specific technology/pattern

## Agent
Solutions Architect, Technical Analyst

## Prompt mẫu
```
/fis:ask "Should we use microservices or modular monolith for a team of 5 developers?"
```
```
/fis:ask "What's the best caching strategy for a real-time leaderboard with 1M users?"
```

## Nguyên tắc
- Provide context (team size, scale, constraints)
- Ask specific questions (not vague)
- Consider trade-offs in answers

## Verification
- [ ] Question answered with reasoning
- [ ] Trade-offs discussed
- [ ] Actionable recommendation provided
