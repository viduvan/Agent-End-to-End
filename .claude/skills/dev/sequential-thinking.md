# Skill: /fis:sequential-thinking — Step-by-Step Analysis

> Phân tích từng bước cho vấn đề phức tạp, có thể revise. Structured reasoning với backtracking.

## Khi nào dùng
- Vấn đề phức tạp cần phân tích từng bước
- Debugging logic issues
- Architecture reasoning
- Decision making với nhiều factors

## Agent
Technical Analyst, Solutions Architect

## Prompt mẫu
```
/fis:sequential-thinking "Analyze why the caching layer is causing stale data. Think step by step."
```
```
/fis:sequential-thinking "Design database schema for multi-tenant SaaS. Consider: isolation, performance, migration."
```

## Nguyên tắc
- Think step by step (chain of thought)
- Cho phép revise nếu phát hiện sai ở bước trước
- Document reasoning tại mỗi step

## Verification
- [ ] Each step clearly documented
- [ ] Reasoning logical and traceable
- [ ] Conclusion follows from steps
