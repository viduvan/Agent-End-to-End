# Skill: /fis:mobile-development — Mobile Development

> Dựng app mobile với React Native, Flutter, Swift, Kotlin. Cross-platform và native development.

## Khi nào dùng
- Build mobile apps (iOS/Android)
- React Native cross-platform development
- Flutter cross-platform development
- Native iOS (Swift) / Android (Kotlin)
- Mobile-specific patterns (navigation, gestures, notifications)

## Agent
Fullstack Developer (Mobile Specialist)

## Prompt mẫu
```
/fis:mobile-development "Create React Native app with bottom tab navigation, auth screens, and profile page"
```
```
/fis:mobile-development "Build Flutter widget for real-time chat with message bubbles and typing indicator"
```

## Nguyên tắc
- Choose platform based on requirements (cross-platform vs native)
- Mobile-first UX patterns (gestures, haptics, safe areas)
- Test on both iOS and Android

## Verification
- [ ] Runs on both iOS and Android (if cross-platform)
- [ ] Safe areas handled
- [ ] Navigation patterns mobile-friendly
- [ ] Performance: 60fps scrolling
