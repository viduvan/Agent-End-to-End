# Skill: /fis:port-feature — Port Feature Between Codebases

> Port feature giữa các codebases. Extract, adapt, integrate code từ repo khác.

## Khi nào dùng
- Port specific feature từ repo A sang repo B
- Adapt open source implementation cho project
- Extract reusable modules từ monolith

## Agent
Fullstack Developer

## Prompt mẫu
```
/fis:port-feature "Port the notification service from project-A into this project"
```

## Nguyên tắc
- Adapt to target project's conventions (naming, structure, patterns)
- Check license compatibility
- Add tests for ported code

## Verification
- [ ] Code adapted to project conventions
- [ ] Tests added
- [ ] License checked
