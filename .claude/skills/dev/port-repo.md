# Skill: /fis:port-repo — Port Entire Repository

> Port toàn bộ repository. Clone, adapt, reconfigure cho context mới.

## Khi nào dùng
- Fork + customize entire repository
- Migrate codebase to new framework
- Create white-label version

## Agent
Fullstack Developer, Solutions Architect

## Prompt mẫu
```
/fis:port-repo "Port this Express.js app to Next.js API routes"
```

## Nguyên tắc
- Understand source before porting
- Incremental migration (not big bang)
- Keep git history where possible

## Verification
- [ ] All features ported
- [ ] Tests pass in new context
- [ ] Documentation updated
