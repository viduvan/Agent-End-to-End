# Skill: /fis:stitch — AI Design-to-Code Pipeline

> Sinh thiết kế UI bằng AI Google Stitch và xuất Tailwind/HTML. Design → Code automation.

## Khi nào dùng
- Convert Figma/screenshot designs thành code
- Generate UI từ text description bằng Google Stitch
- Rapid prototyping UI components
- Export Tailwind CSS / HTML / React code

## Agent
UI/UX Designer, Fullstack Developer

## Prompt mẫu
```
/fis:stitch "Generate landing page UI from this description: dark theme, hero with 3D animation, features grid, pricing table"
```

## Verification
- [ ] Generated code matches design intent
- [ ] Code uses semantic HTML
- [ ] Responsive layout
