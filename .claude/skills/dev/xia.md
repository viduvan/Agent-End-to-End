# Skill: /fis:xia — Cross-Repo Feature Porting

> Trích xuất, so sánh, port feature từ repo GitHub vào project. Cross-repository feature adoption.

## Khi nào dùng
- Port feature từ open source repo vào project
- Học implementation patterns từ popular repos
- Compare implementation approaches giữa repos
- Extract reusable modules

## Agent
Fullstack Developer

## Prompt mẫu
```
/fis:xia "Port the auth middleware from https://github.com/example/app into our project"
```
```
/fis:xia "Compare how Stripe webhook handling is done in 3 popular open source projects"
```

## Nguyên tắc
- Adapt, don't copy — fit project conventions
- Check licenses before porting
- Understand code before adopting

- [ ] Ported code adapted to project conventions
