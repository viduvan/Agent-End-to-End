# Skill: /fis:copywriting — Copywriting Patterns

> Công thức copywriting chuyển đổi và pattern email/headline. Content marketing, landing page copy, email sequences.

## Khi nào dùng
- Viết landing page copy (headlines, CTAs, features)
- Email marketing sequences (welcome, nurture, re-engagement)
- Product descriptions và value propositions
- Social media content

## Agent
Content Specialist

## Prompt mẫu
```
/fis:copywriting "Write landing page headline and subhead for AI code review tool targeting senior developers"
```
```
/fis:copywriting "Create 5-email welcome sequence for SaaS product trial users"
```
```
/fis:copywriting "Generate 10 headline variations for A/B testing on pricing page"
```

## Nguyên tắc
- Benefit-focused (not feature-focused)
- Clear CTA in every piece
- A/B test copy variations

## Verification
- [ ] Headlines benefit-focused
- [ ] Clear CTA present
- [ ] Tone matches brand
